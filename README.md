# CloudDrive 云盘 — 静态演示站点

一个仿微软 OneDrive 风格的纯静态 HTML 云盘演示网站。无需服务器，直接用浏览器打开即可运行。

## 页面

| 文件 | 说明 |
|------|------|
| `login.html` | 登录页（邮箱 / 密码 / 企业 SSO） |
| `index.html` | 我的文件（列表 / 网格视图，上传、新建文件夹、共享、重命名、删除） |
| `recent.html` | 最近使用的文件 |
| `shared.html` | 与我共享的文件 |
| `photos.html` | 照片库（按月分组网格视图） |
| `trash.html` | 回收站（还原 / 永久删除 / 清空） |
| `css/styles.css` | 全局样式 |
| `js/app.js` | 交互逻辑（演示数据、模态框、Toast、右键菜单、详情面板） |

## 快速预览

直接在浏览器中打开 `login.html` 即可体验，或用任意静态文件服务器托管：

```bash
python3 -m http.server 8080
# 然后访问 http://localhost:8080/login.html
```

## 功能亮点

- ✅ 完全纯静态，零依赖，零构建步骤
- ✅ 列表 / 网格双视图切换（偏好持久化）
- ✅ 文件右键菜单（下载、共享、重命名、复制、删除）
- ✅ 详情面板（属性、快捷操作）
- ✅ 模态框：新建文件夹、上传（拖放区域）、共享链接、重命名、删除确认
- ✅ Toast 通知
- ✅ 侧边栏存储用量指示器
- ✅ 响应式布局（移动端适配）
